#  Login Authentication System (Frontend)

##  About
This is a **frontend project** for a Login & Signup system built using React.  
It includes user registration, login, and dashboard navigation.

##  Features
- User Signup with email & password
- User Login with validation
- Dashboard page after login
- Simple & clean UI with React + CSS

##  Tech Stack
- React JS
- HTML / CSS
- JavaScript
